# Website Ágil Serv - Assistência Técnica

Site institucional para a empresa Ágil Serv, especializada em assistência técnica de máquinas de lavar e geladeiras em Goiânia-GO.

## Estrutura do Projeto

```
Website-agilservgo-using-Claude-Code/
├── index.html              # Página principal do site
├── css/
│   └── style.css           # Estilos customizados
├── js/
│   └── main.js             # Interações JavaScript
├── Fotos/
│   ├── agilserv_logo.png   # Logo da empresa
│   └── site_antigo.png     # Screenshot do site anterior
├── contexto                # Arquivo de contexto do projeto
└── README.md               # Este arquivo
```

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS)
- JavaScript (Vanilla)
- Font Awesome (ícones)
- Google Fonts (Poppins)

## Funcionalidades

- Design responsivo (mobile-first)
- Menu hamburger para dispositivos móveis
- Scroll suave para navegação
- Botão flutuante do WhatsApp com animação
- Animações de entrada nos elementos

## Seções do Site

1. **Header** - Logo e navegação
2. **Hero** - Chamada principal com CTA para WhatsApp
3. **Diferenciais** - Cards com os benefícios da empresa
4. **Serviços** - Máquinas de lavar e geladeiras
5. **Sobre Nós** - História da empresa
6. **Depoimentos** - Avaliações de clientes
7. **Contato/Footer** - Informações de contato

## Como Executar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Ou use um servidor local como Live Server (extensão VS Code)
